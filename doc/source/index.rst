Welcome to rpOptBioDes's documentation. This tool provides a docker that extends the original doebase_ project to parse rpSBML_ files and an SBOL_. The original projects explores the design space of combination of promoters, coding sequences, terminators and RBS of the heterologous pathway generated by RetroPath2.0_.

.. code-block:: bash

   docker build -t brsynth/rpoptbiodes-standalone .

You can run the docker using the following command:

.. code-block:: bash

   python run.py -input test/test_rpGlobalScore.tar -input_format tar -input_sbol test/test.sbol -output test/test_output.tar
